# Digital Literacy Portfolio 🌐
**Student Digital Ambassador Project | VIT Bhopal University**

## 📌 Project Overview
This repository documents my journey as a **Student Digital Ambassador**. The project focuses on building a professional online identity, mastering digital communication, and promoting cyber safety within the academic community.

## 📂 Project Structure

### [Task 1] Digital Literacy Awareness Infographic
*   **Tools:** Canva
*   **Content:** A visual guide on the core pillars of digital literacy, safe internet practices, and email etiquette.
*   **Goal:** Helping peers understand responsible technology use.

### [Task 2] Student Digital Portfolio
*   **GitHub:** Profile README with academic goals and branch details.
*   **LinkedIn:** Professional networking and education updates.
*   **HackerRank:** Tracking technical skill development and coding progress.

### [Task 3] Coding & Collaboration Platforms
*   **Python:** Completion of beginner-level challenges on HackerRank.
*   **Collaboration:** A "Digital Literacy Awareness Quiz" created via Google Forms, integrated with Google Sheets for data management.

### [Task 4] Professional Email & Etiquette
*   **Communication:** Templates for academic extensions and internship inquiries.
*   **Social Media:** A "Do’s and Don’ts" checklist emphasizing the "Billboard Test" for digital footprints.

### [Task 5] Cybercrime Awareness & Prevention
*   **Case Study:** Analysis of UPI/Online Payment Fraud (e.g., "Pending Utility Bill" scam).
*   **Action Plan:** An 8-point "Stay Safe Online" guide and reporting resources.
